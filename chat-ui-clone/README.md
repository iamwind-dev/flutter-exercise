A modern and responsive Flutter chat application with a clean UI inspired by Messenger and WhatsApp. This app features local message management, auto-replies, and smooth animations.

## Features

- **Modern UI Design** - Clean and intuitive chat interface
- **Real-time Messaging** - Send and receive messages instantly
- **AI Bot Integration** - Automatic contextual replies from AI Bot
- **Timestamps** - Display time for each message
- **Message Bubbles** - Different styles for user and bot messages
- **Responsive Design** - Works on all screen sizes
- **Smooth Animations** - Typing indicators and scroll animations
- **Profile Header** - Avatar, name, and online status
- **Smart Input** - Auto-focus management and send button state

## Getting Started

### Prerequisites

- Flutter SDK (3.9.2 or higher)
- Dart SDK
- Android Studio / VS Code
- Android Emulator / iOS Simulator / Physical Device

### Installation

1. **Clone the repository**

   ```bash
   git clone <repository-url>
   cd chat_ui_clone
   ```
2. **Install dependencies**

   ```bash
   flutter pub get
   ```
3. **Run the app**

   ```bash
   flutter run
   ```
